= Limitaciones en NetApp Copy and Sync
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Las limitaciones conocidas identifican plataformas, dispositivos o funciones que no son compatibles con esta versión del producto o que no interoperan correctamente con él. Revise estas limitaciones cuidadosamente.

NetApp Copy and Sync no es compatible con las siguientes regiones:

* Regiones gubernamentales de AWS
* Regiones gubernamentales de Azure
* Porcelana
